# Augment README Creator

A VS Code extension that uses the Augment AI extension to automatically generate README files for your projects.

## Features

- Automatically generates comprehensive README files for your projects
- Uses Augment AI to create context-aware content
- Customizes the README based on your project's package.json information
- Provides options to create a template or copy from Augment

## Requirements

- VS Code 1.82.0 or higher
- Augment extension must be installed and configured

## Usage

1. Open a project folder in VS Code
2. Run the command "Create README with Augment" from the command palette (Ctrl+Shift+P)
3. The extension will analyze your project and use Augment to generate a README
4. Choose to create a template or copy the content from Augment

## Extension Settings

This extension does not contribute any settings.

## Known Issues

- The extension cannot directly capture Augment's response, so manual copying is required
- There may be a delay while Augment processes the request
